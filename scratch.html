Scratch

Name Assignment

Description:

I used scratch to design and create a program that shows my name. The requirements were to have a sprite for each letter of my name and have them all have 3 unique behavior that started when the green flag was pressed. For extra credit, if u press the spacebar it will reset.

Concepts Learned

  • Using event blocks (when green flag is clicked, when key is pressed…)
  • Conditional statements ( if/then statements)
  • Used loops ( forever block/repeat block)
  • Sprites (created my own sprites with sprite editor, chose from the library of sprites, changed costumes)
  • Movement (used x and y values to make sprites move to certain locations, rotated sprites)

Final Project

Description:

I had to program an interactive game using scratch where the user would bounce a ball and break out pieces. Once all of the pieces were broken the second level began, and once those were gone, the game would end. There had to be a timer, counter and a help button for how to play the game.

Concepts Learned

  • Using event blocks (when green flag is clicked, when key is pressed…)
  • Sprites (created my own sprites with sprite editor, chose from the library of sprites, changed costumes)
  • Movement (used x and y values to make sprites move to certain locations, rotated sprites)
  • Say blocks (to make lines of script)
  • Hide/Show blocks (to make sprites disappear and reappear) <.li>
  • Conditional statements ( if/then statements if a sprite touched another it would disappear, bounce, etc)
  • Used loops ( forever block/repeat block for constant movement, players ball always bounces off of the edges of the screen)
